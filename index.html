Motion Simulating Robot Through Hand Gestures

Project Brief:

It is a real time monitoring system based on IoT, by which humans can interact with robots through hand gestures. This is an immense aid for people for whom mobility is a great challenge. Gesture recognition has always been a technique to decrease the distance between the physical and the digital world. In this work, we introduce an IoT based vehicle system which no longer requires manual controlling of the robot. The proposed work is achiebed by utilizing the NodeMCU microcontroller. In addition, the system is stocked with an IP Camera for various monitoring purposes; also it has an automatic obstacle detection system which is introduced to improve the safety measurements to avoid any hazards.
